## Installation

1 Requirements

* [Node.js](https://nodejs.org/en/)
* [npm](https://www.npmjs.com/)

2.1 Clone the repository

```bash
git clone https://github.com/marcelo-schreiber/portfolio
```

2.2 Install dependencies

```bash
npm install
```

2.3 Start the development server

```bash
npm run dev
```
